## ğŸ’¡ Usage Tips

### 1. Add Animations
Simply add animation classes to any element:
```html
<div class="animate-fade-in-up">
    This fades in while sliding up
</div>
```

### 2. Combine Effects
Mix animations with hover effects:
```html
<div class="card animate-fade-in hover-lift">
    Fades in and lifts on hover
</div>
```

### 3. Stagger Animations
Create sequential animations:
```html
<div class="card animate-fade-in-up" style="animation-delay: 0s;">1</div>
<div class="card animate-fade-in-up" style="animation-delay: 0.1s;">2</div>
<div class="card animate-fade-in-up" style="animation-delay: 0.2s;">3</div>
```

### 4. Custom Timing
Override animation duration:
```html
<div class="animate-bounce" style="animation-duration: 0.5s;">
    Faster bounce
</div>
```
